# Read Controller Registers

## Security-relevance

The capability for someone on the host to read registers can aid in the creation of a system to perform debugging-like actions on the BT Controller. A debugging-like system can aid in the development of reverse engineering and/or creation of over-the-air memory-corruption exploits, as in [1].
